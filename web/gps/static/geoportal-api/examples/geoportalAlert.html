Utilisation d'une carte Géoportail pour de la saisie d'évènements simples comme la localisation d'une anomalie, d'une information, etc ...

Vous pouvez envoyer votre anomalie à geop_.nosp@m.autres@geo.nosp@m.portail.fr !

Comment saisir une anomalie ?

  1. On crée une nouvelle couche via le bouton . Le choix d'une couche vectorielle dépend de l'anomalie ;
  2. Un nom peut être donné à la couche ;
  3. Pour les couches linéaires ou surfaciques, il est possible de dessiner en déplaçant la souris la géométrie des objets (mode à main levée) ;
  4. Il est possible d'importer un fichier KML, une trace GPX ou un export OpenStreetMap dans la visualisation. Expérimental
  5. Une fois ces choix effectués, on clique sur Ajouter. La couche apparaît alors dans le gestionnaire en haut, à gauche ;
  6. La saisie elle-même passe par l'utilisation de l'outil dessin ( pour les couches ponctuelles, pour les couches linéaires, pour les couches surfaciques) ;
  7. À la fin de la saisie des objets vectoriels, une fenêtre apparaît pour saisir le nom de l'anomalie et une description ;
  8. Une fois les informations fournies, il suffit de fermer la fenêtre ;
  9. Il est possible d'effectuer plusieurs saisies, sur plusieurs couches ;
  10. Pour quitter le mode saisie, il suffit de cliquer sur ;
  11. Une fois la saisie terminée, plusieurs outils permettent de la reprendre :
    1. permet de déplacer un objet saisi ;
    2. permet de changer la géométrie des objets par déplacement des points des objets vectoriels ;
    3. permet de détruire l'objet qui va être sélectionné avec la souris ;
    4. permet de sélectionner un objet. La sélection ouvre la fenêtre de saisie du nom et de la description de l'anomalie (à minima, d'autres attributs ayant pu être ajoutés) ou de la fiche associée au KML/GPX/OSM.
  12. Il est possible d'agir sur la couche en cliquant sur
              
    pour accéder aux outils suivant :
    1. suppression de la couche en cliquant sur ;
    2. ajout d'attributs sur la couche en cliquant sur ;
    3. changement de la couleur des objets (coutours et remplissage) en cliquant sur ;
    4. sauvegarde sur votre poste de travail de la couche saisie (et éventuellement de nous l'envoyer par courriel) en cliquant sur . Seuls les exports GML, GPX, OSM et Géoconcept sauvegardent les attributs additionnels (autre que nom et description) ;
    5. zoom sur l'emprise de la couche en cliquant sur .

Comment saisir un objet ?

  1. La saisie d'un point débute par un clic sur , puis chaque clic gauche sur la carte permet de localiser un objet ponctuel. La fiche de l'objet s'ouvre alors et il suffit de rentrer les valeurs pour le champ Nom (le titre de la fiche) et sa description (le corps de la fiche). L'ajout d'attributs additionnels via permet d'avoir ces attributs automatiquement dans la fiche ;
  2. La saisie d'un ligne débute par un clic sur , puis un clic gauche débute la saisie de la ligne, un trait relie alors le point saisi avec la position de la souris. Un autre clic gauche saisit un nouveau point et ainsi de suite. Pour terminer, un double-clic gauche est nécessaire. Lors de ce double-clic, la fenêtre de saisie s'affiche. Le mode à main levée est possible sur les saisie de ligne à la création de la couche ;
  3. La saisie d'une surface débute par un clic sur , puis un clic gauche débute la saisie du polygone, un trait relie alors le point saisi avec la position de la souris. Un autre clic gauche saisit un nouveau point et ainsi de suite. Pour terminer (et donc fermer le polygone), un double-clic gauche est nécessaire. Deux actions sont associées à ce double-clic : la saisie de l'avant dernier point du polygone et la fermeture du polygone (le dernier point du polygone est donc le premier saisi). Il ne reste plus qu'à remplir la fiche. Le mode à main levée est possible sur les saisie de ligne à la création de la couche.
